# Spinotek x HIMA TI Politeknik Hasnur | Hackathon Repository

Selamat datang di repositori resmi **Spinotek x HIMA TI Politeknik Hasnur**. Repositori ini digunakan untuk menampung dan memamerkan hasil karya peserta program **Skill Upgrading & Hackathon**.

Website utama akan secara otomatis menampilkan daftar proyek yang ada di dalam repositori ini berdasarkan konfigurasi yang terdaftar.

### 1. Fork & Clone

1. **Fork** repositori ini ke akun GitHub kamu.
2. **Clone** ke komputer lokal kamu:
   ```bash
   git clone https://github.com/USERNAME_KAMU/spinotek-hima-ti-polhas.git
   cd spinotek-hima-ti-polhas
   ```

### 2. Tambahkan Folder Proyek

1. Buat folder baru di **root** (halaman awal) folder proyek. Gunakan nama folder yang unik dan relevan (gunakan _kebab-case_, huruf kecil semua, pisahkan dengan strip).
   - Contoh: `sistem-absensi-online`
2. Di dalam folder tersebut, pastikan ada file `index.html` sebagai halaman utama proyek kamu.
3. Struktur folder kamu harus terlihat seperti ini:

### 3. Simpan & Push

1. Simpan semua perubahan.
2. Lakukan commit dan push ke repositori fork kamu:
   ```bash
   git add .
   git commit -m "feat: add project nama-proyek-kamu"
   git push origin main
   ```

### 4. Submit Pull Request (PR)

1. Buka repositori asli di [Spinotek-Organization/spinotek-hima-ti-polhas](https://github.com/Spinotek-Organization/spinotek-hima-ti-polhas).
2. Pergi ke tab **Pull Requests**.
3. Klik tombol **New Pull Request**.
4. Pilih cabang dari fork kamu dan arahkan ke branch **`main`** di repositori asli.
5. Berikan judul dan deskripsi PR yang jelas.
6. Klik **Create Pull Request**.
